# Hello World – Multi-Language Collection

This repository is a personal collection of **“Hello, World!” programs written in many different programming languages** — ranging from modern, popular languages to older and more niche ones.

The goal of this project is simple:
-  Learn and compare language syntax
-  Understand how different ecosystems structure their entry point
-  Build a growing reference of language basics

---

## Structure

Each file contains:
- A **comment at the top** stating the language
- A minimal **Hello, World!** implementation
- A consistent filename format (`main.ext` or language-standard equivalent)

## How to Use

You can:
- Open files directly to compare syntax
- Compile or run them using the appropriate compiler/interpreter
- Use this as a reference when learning a new language
- Expand it with your own additions

---

##  Notes

- This repository is **not meant to be production code**
- Some languages may require specific compilers or runtimes
- Assembly examples may be platform-specific

---

##  Contributing

If you’d like to add another language:
1. Follow the existing naming convention
2. Include a comment at the top: `Hello World in <Language>`
3. Keep it minimal and readable

Pull requests are welcome.

---

##  Why This Exists

Because *“Hello, World!”* is where every programmer starts —  
and it’s cool to see how many different ways humans have invented to say it.
